About me

Hey! My name's Jon and I build applications for the modern web. Growing up around the Charlotte, NC area, I discovered my love for all things technology early on. At the age of 10 I had my first taste of coding when I aimlessly edited some values in a Java file while trying to create my own server for the game RuneScape. After that I would go on to mess around on and off with programming throughout my childhood. Creating games, desktop applications, and websites.

After graduating high school, I followed through on my plan of becoming a self-taught web developer. Using a casual mix of books, real-life mentors, and an abundance of online resources, I got serious about developing my skills. The project's I've created during this time can be seen below!
