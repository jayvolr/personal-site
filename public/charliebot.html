About this project

CharlieBot is a chatbot for Discord and was my first creation in Node.js when I was just starting to learn bank end programming.

CharlieBot is not a web app, so I wasn't able to depend on a framework like Express to build the project. This really helped me realise the capability of Node.js on its own, and later on I was able to appreciate how useful Express.js is when creating web apps.

Technologies used

  • Ubuntu (desktop and server)
  • Node.js
  • Discord.js
